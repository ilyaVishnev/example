Запуск происходит из командной строки , где в качестве аргументов вводятся имя входящего и исходящего файла через пробел .

Выходной файл будет содержать предупреждения :
- если командная строка не содержит нужного кол-ва аргументов
- если разрешения файлов не соответствуют необходимым
- если входной файл отсутствует


Входной файл должен содержать строки 6 координат точек равнобедренного треугольника через пробел.
Выходной файл должен содержать координаты максимального по площади равнобедренного треугольника.

Выходные данные не будут содержать координаты макс. треугольника , если :
-входные данные содержат несколько равнобедренных треугольников с одинаковой макс. площадью

В выходной файл добавляются предупреждения о допущенных ошибках , если :
-количество координат не равно 6
-координаты содержат буквы

Из расчета макс. площади выпадают координаты , если:
-количество координат не равно 6
-координаты содержат буквы
-входные данные содержат координаты Не равнобедренных треугольников
